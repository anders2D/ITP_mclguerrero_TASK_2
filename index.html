¡Hola! Estás viendo Striped, una plantilla de sitio web HTML5 completamente receptiva diseñada por AJ para HTML5 UP. Cuenta con un diseño limpio y minimalista, estilos para todos los elementos básicos de la página (incluyendo citas, tablas y listas), una barra lateral reposicionable (izquierda o derecha), y código HTML5/CSS3 diseñado para personalización rápida y sencilla (consulta los comentarios del código para obtener detalles).

Striped se ofrece de forma gratuita bajo la licencia de Atribución de Creative Commons, así que siéntete libre de usarlo en proyectos personales o incluso comerciales, solo asegúrate de dar crédito a HTML5 UP por el diseño. Si te gusta lo que ves aquí, asegúrate de visitar HTML5 UP para ver más diseños geniales o seguirme en Twitter para conocer nuevas versiones y actualizaciones.
